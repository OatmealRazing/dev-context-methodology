# Stage 03: Setup

## Inputs

| Source | Location | Why |
|--------|----------|-----|
| Planning summary | ./stages/02-planning.md | Defines stack, architecture, milestones |

## Process

1. Create project directory and initialize repository
2. Set up remote repository
3. Write agent instructions (CLAUDE.md or equivalent)
4. Scaffold project structure per the implementation plan
5. Install dependencies and verify tooling works
6. Confirm the project builds and runs (even if empty)
7. Make initial commit
8. Register project in tracker (if not already tracked)

## Outputs

| Artifact | Location | Format |
|----------|----------|--------|
| Setup summary | ./stages/03-setup.md | Repo URL, scaffold notes, tooling confirmed |

## Gate — Exit Criteria

- [ ] Repository created with remote
- [ ] Agent instructions written
- [ ] Project scaffold in place
- [ ] Dependencies installed and tooling verified
- [ ] Project builds/runs successfully
- [ ] Initial commit pushed
