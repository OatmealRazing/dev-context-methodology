# Definition of Done

## Required -- All Items

- [ ] Acceptance criteria verified
- [ ] No known regressions introduced
- [ ] Changes committed to main branch

## By Item Type

### Feature

- [ ] User-facing docs updated (if applicable)
- [ ] Works across supported platforms

### Bugfix

- [ ] Root cause identified and noted
- [ ] Regression test added

### Release

- [ ] Version consistent across all manifest files
- [ ] Changelog accurately covers all user-facing changes
- [ ] Clean build at tagged commit

### Refactor

- [ ] All existing tests still pass (behavior preserved)
- [ ] Code is clearer than before, not just different
- [ ] Only the agreed refactor scope was changed

### Spike

- [ ] Findings documented
- [ ] Go/no-go recommendation written

### Chore

- [ ] No additional criteria beyond Required
